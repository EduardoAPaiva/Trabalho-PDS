# Processamento Digital de Sinais – Projeto Final (Grupo A4)

Este repositório contém a implementação desenvolvida para o Projeto Final da disciplina **SEL0615 – Processamento Digital de Sinais**.

O projeto consiste na implementação da **Transformada de Taylor-Fourier (TTF)** para análise espectral de sinais de corrente elétrica e na detecção do instante de inserção de cargas não lineares a partir da resposta da transformada.

## Grupo

**A4**

## Arquivo

* `taylor-fourier.py` – Definição e execução da Transformada de Taylor-Fourie. Escolher manualmente no código a ordem para o polinômio de taylor e o tamanho em ciclos para cada uma das janelas.

## Execução

Execute o script principal após inserir os arquivos de sinal na pasta `sinais`. O sinal utilizado por padrão é `sinal_4_semruido.csv`
